Sandpack 是一个用于创建实时运行的代码编辑体验的组件工具包，由 [CodeSandbox](https://codesandbox.io/) 上使用的在线捆绑器提供支持。

它提供了一个开放的组件和实用程序生态系统，让您可以在浏览器中编译和运行现代框架。您可以使用我们的预定义组件之一将 CodeSandbox 体验嵌入到您的项目中，也可以在我们的标准组件和实用程序之上构建您自己的沙包版本。

随着您对文档的逐步了解，您将更深入地了解我们的生态系统。
